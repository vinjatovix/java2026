# Hello World in Java

Minimal project that prints `Hello, World!` to the console.

## Requirements
- JDK 17
- Terminal

## Compile and run
The class is in the package `com.vinjadevix.lesson01`. The source file is `src/com/vinjadevix/lesson01/Main.java`.

Compile (single file):
```
javac -d out src/com/vinjadevix/lesson01/Main.java
```
Run:
```
java -cp out com.vinjadevix.lesson01.Main
```

## Suggested structure
- `src/` \- source code organized by package
- `out/` \- compiled classes
